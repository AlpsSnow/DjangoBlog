1. 创建python虚拟环境  
2. pip install django  
3. python -m django --version  
4. django-admin startproject DjangoBlog  
5. cd DjangoBlog  
6. tree /F
``` 
├─DjangoBlog
│  │  settings.py
│  │  urls.py
│  │  wsgi.py
│  │  __init__.py
│─manage.py
```
7. python manage.py runserver  
8. http://localhost:8080/admin/
